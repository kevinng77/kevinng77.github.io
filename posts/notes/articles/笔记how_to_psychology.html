    # 0. 前言

    相关信息

    心理学涉及人类行为和意识的不同形式，可以用来解释、预测和控制人类行为。

    心理现象和心理问题与每个人息息相关，人们试图通过心理学解释这些问题和现象，并从中获得行之有效的帮助和建议。在这方面 精神分析 和治疗技术有一定的优势，而其他大多数心理学研究针对的则是群体的普遍行为规律，偏重与解释和预测，齐研究结论都具有概率性和领域特异性，因此无法有针对性地、面面俱到地解决个体所有的心理问题。

    心理学的许多基础研究，确实与大众生活有一定距离。

    这本书的初衷：对批判性思维技能进行简要介绍，帮助人们更好的理解心理学的主题。

    # 1. 心理学在科学阵营中的表现

    1. 弗洛伊德的方法（精神分析）不能代表现代心理学。

    心理学统一的特征：用科学的方法寻求对行为的理解。但精神分析包含理论和假设，缺乏实证观察（难复现）。

    1. 心理学分支众多，似乎难以统一

    心理学采用科学的方法，探究 人类和动物的所有行为。他是基于数据的科学的行为研究。

    如果用当前所掌握的实证方法，某个问题无法解决或某个理论不可检验，那么科学家们就不会对它展开研究。

    # 2. 可证伪性

    一个假设被确信为真的程度，与该假设是否为真无关。

    当一种理论过度自我保护以免于被证伪时，它就根本不再被认为是科学的。

    本章举例对精神分析的一些观念做出了批评，指明精神分析中不可证伪理论的存在会对社会造成危害。书中以阻碍综合征例子展开了说明。

    本书也提倡每个科学家都应具备证伪的态度，让科学发挥作用。

    注意

    真正的危险是人类有一种固有的倾向，即避免让我们的信念暴露在可能被证明是错误的情境之中。

    # 3. 操作主义和本质主义

    # 3.1 操作主义

    本质主义认为，只有从内在本质或者本质属性的角度对现象做出终极解释的理论，才算得上是好的科学理论。

    科学不去回答这种本质主义问题，而是靠对概念进行操作性定义得以进步的。

    操作主义 就是这样一种思想： 科学理论的概念必须以某种方式建立在可观察事件的基础之上，或与之相关联，而这些可观察事件是可以被测量的。

    信度：测量工具（操作性定义）的一致性

    结构效度：一个测量工具（操作性定义）是否测量了它本应测量的内容。

    测量需要具备高效度和高信度。 一个有信度而没有效度的例子：测量者通过量脚码来测你的智力。

    # 3.2 预设偏见

    我们每个人都有关于人格和人类行为的直觉理论，因为我们在生活中一直在向自己“解释”自己和他人的行为。我们所有的个人心理学理论中都包含着理论概念（例如聪明、攻击和焦虑）。心理学中的许多专业概念都是用日常语言中的词汇命名的，这是误解产生的最大根源之一，也是在媒体上准确地呈现心理学发现的最大障碍之一。

    外行人很少意识到，当心理学家把“智力”“焦虑”“攻击”“依恋”等词语用作理论构念时，他们所指的含义不一定与大众使用这些词汇时的含义相同。

    # 4. 个案研究

    针对某个现象的孤证具有高度的误导性。见证叙述可能成为无用的证据。

    # 4.1 安慰剂效应

    安慰剂效应的控制：在测试新药时候，两组病人都不知道他们吃的是什么药。这样当比较两组的结果时，安慰剂效应（即给病人任何一种新的治疗都会使他们感觉更好的倾向）就得到了控制。

    所有类型的心理治疗都涉及安慰剂效应，许多有轻度或中度心理问题的人，在接受心理治疗后说他们的情况有所好转。

    相关信息

    人们很容易相信见证叙述，他们很容易理解。对他们进行批判性思考需要付出努力，而许多人不愿意做出认知上的努力。

    # 4.2 鲜活性

    当面临问题解决或决策情境时，人们会从记忆中提取出当前情境有关的信息。因此，人们倾向于利用更容易得到的、能够用来解决问题或做出决策的信息。强烈影响可获得性的一个因素，就是信息的鲜活性。

    没有什么比真诚的个人见证更鲜活。一个例子就是：我们在购物前仔细收集了不同品牌的大量信息，最后却由于某个朋友的推荐，而在最后一刻放弃了自己的选择。

    除了朋友外，媒体也能制造鲜活性。

    1 型加工：处理一个问题时使用最不费力的心理加工过程。

    # 4.3 巴纳姆效应

    巴纳姆效应： 绝大多数人会认为繁华的个性 总结是对自己准确而独特的描述。巴纳姆效应是人们对手相学和占星术的准确性坚信不疑的基础。

    通过巴纳姆效应，我们也能知道用见证来支持某一特定的主张是毫无意义的。

    相关信息

    大部分涉及到巴纳姆效应的伪科学行业油水颇丰，如占星等。以治疗关节炎为例，用来识别不道德推销者的方法：

    1. 他们会提供一种用于治疗关节炎的特别方法或设备。
    2. 他会做广告，用的都是个案和来自满意患者的见证。
    3. 他们会承诺这是快速的简单的方法。
    4. 他们会声称知道关节炎的原因，并谈到清楚体内毒素，促进健康。他们会说外科手术、X 光和医生开的处方是没有必要的。
    5. 他可能会指责医疗机构阻扰进步，但是却不让人们用验证方式对他的方法加以验证。

    # 5. 相关和因果

    留意选择偏差的存在，避免只根据相关数据来做出因果推论。

    # 5.1 第三因素

    两个有相关性的因素，可能没有因果关系。两个因素相关，可能与但三个未被测量的变量相关。在某些情况下，只要相关就够了，例如我们的目标是预测而不是决定原因。

    比如溺水人数和吃冰淇淋人数呈相关性，但强制不卖冰淇淋并不会减少溺水人数。

    # 5.2 因果关系的方向性

    确定因果关系的方向性是个常见的问题。

    有研究表明参加志愿者工作的人比不参加志愿者工作的人更幸福。在排除了第三变量之后，需要确定因果关系方向。

    采用第六章真实验的逻辑，验证了幸福让人们更利他，而且利他行为也让人更幸福。

    如户外食品广告越多的人口普查区，人们越肥胖。我们可能会想到，广告商可能会在食品的高消费社区投放更多的广告。

    # 5.3 自我选择偏差

    自我选择偏差指的是人们自己选择进入一个特定的群体，而不是被随机分配的情况。

    自我选择会产生个人变量和环境特征之间的虚假关系。如某个州空气最好，但在这个州里，由呼吸系统疾病导致的死亡率高于国家平均水平。人们可能误导是这个州的空气害死了这些人。但或许这个州的空气的确很干净，患有呼吸系统疾病的人都喜欢搬到那里。

    自我选择偏差其实也是因果关系方向的一个例子。

    比如下面这个观点：大量研究发现，适度饮酒者不仅比经常饮酒者健康，而且比不饮酒者健康。如果我们不知道是适度饮酒导致了积极的健康结果，还是适度饮酒群体的一些良好特征导致了这一结果，我们就不能说适度饮酒就是原因。

    # 6. 科学实验

    试验方法的核心是操纵与控制。

    # 6.1 随机分配

    随机分配与操纵共同定义了真实验研究。

    当我们无法控制所有干扰因素时候，采用重复的大规模随机分配实验可以消除系统误差。

    随机样本与随机分配不是一回事。随机取样是指如何选择被试进行研究。随机分配是指，在真实验中，实验人员会设置实验组和控制组，当每一名被试被分到实验组或控制组的机会相等时，就实现了随机分配。

    # 6.2 描述性事实与假设

    当从描述性事实过快地跳到特定假设解释，大概率会出现错误。

    # 6.3 直觉心理学

    类似于物理学，尽管人们对物体和下落物体有丰富的经验，但他们对运动的直觉理论是非常不准确的，经常不能预防这些直觉性错误。例如，即使是经验丰富的出租车司机，在车速和车程的判断上，还是会犯很多与非职业司机相同的错误。

    对于心理学，个人经验并不能保证不会对人类心理学产生错误的信念。

    人们直觉行为理论的诸多不足之处说明了为什么我们需要有控制的心理学实验。

    # 7. 人为性与心理学

    理论研究与应用研究的异同

    大多数心理学的目的并不是为了直接应用，而是为了发展理论。大多数研究的结果只是通过对理论的修正而间接地得到应用，这些理论再与其他科学规律一起应用于一些实际问题。

    如注意力有限理论，可以用来预测手机带来的交通问题。

    实验室与现场研究结果的一致性程度最高的领域是工业组织心理学，而社会心理学则要低得很多。

    # 8. 聚合性证据的重要性

    科学的两个原则：关联性原则和聚合性证据原则。

    在评估心理学的时证证据时，心中要想的是科学共识，而不是重大突破；是渐进整合，而不是大飞跃。

    # 8.1 关联性原则

    科学中的概念变化遵从关联性原则，新的科学理论不仅要解释新的事实，还要兼容旧的事实。

    心理学也是如此。如果一个新理论否认经典条件作用和操作性作用的存在，它就不可能再心理学中发展起来，因为它与行为科学中的其他知识无法建立联系。

    # 8.2 跃进模式与渐进整合模式

    科学进步不全是通过一次飞跃完成的，更多时候是在多年累积的理论推测、争论和批评后，才达成的新共识。

    许多媒体的报道中，总是存在跳跃性的科学进展。

    如对于阅读和阅读障碍心理学，不同媒体可能有不同的文章报道：

    • 《阅读障碍：阅读困难的孩子有了新希望》
    • 《解开阅读障碍之迷》
    • 《客服阅读障碍：最新脑科学揭示了什么》
    • 《电击有助于阅读障碍儿童提高阅读速度》

    它们都不是治愈方法，而是阅读障碍领域正在取得的缓慢进展中的一部分。

    媒体对此类研究的过早炒作被讽刺地称为“新闻缺陷障碍”。

    # 8.3 聚合性证据

    聚合性证据原则提示我们去审查相关研究文献中缺陷所呈现出的模式和规律，因为这一模式的性质要么支持、要么削弱我们想要得出的结论。

    比如如果所有实现的局限性都不相同，但都能得出同一个实验结果，那么我们会对结论更有信心。（和随机选择有点相似）

    从理论角度解释聚合性证据：当一系列实验一致支持某个理论，同时又共同地排除那些最重要的竞争理论时，研究就具有高度的聚合性。

    比如文章 1 检验了 A、B 和 C 三种实验，然后文章 2 检验了 C、D 和 E 三种实验，我们就有了支持 C 理论的聚合性证据。

    心理学实验的诊断性往往较低，支持某个理论的数据经常只能排除一小部分替代解释，同时保留了许多理论作为可行的候选。只有收集大量的研究数据进行比较之后，才能得出强有力的结论。

    相关信息

    部分媒体可能会利用聚合性证据进行反击：如许多研究结果的聚合证明，观看暴力节目似乎确实增加儿童做出攻击性行为的可能性。电视网络高管可和视频游戏行业可以挑选出个案研究来进行批评，并暗示这些研究都存在缺陷，不足以支持总体结论。

    # 9. 多重原因的问题

    我们证明了行为的一个原因，并不意味着我们已经发现了引发该行为的唯一原因，甚至是最重要的原因。

    在考察行为的原因时，要依照多重原因的原则来思考。预设偏见常常使人们忽略多重原因。

    # 9.1 相互作用

    影响行为的一个因素，会因为另一个因素的存在与否，而产生不同的效果。

    如转校、青春期发育、住所迁移、家庭破裂，没有单一的元素会对学业成绩产生巨大的影响，但当这些生活变化中的几个因素结合在一起时，就会导致学业成绩大幅下降。

    # 10. 概率推理

    心理科学揭示的几乎所有事实和关系都是用概率来表述的。

    人们常常以身边“某某人”作为反驳一个概率观念（如吸烟增加肺癌的概率）的事实， 人类在概率推理这方面表现如此薄弱，以至于海旅推理称为人类理性的操作性定义的核心。

    在所有学科中，心理学确实对人类概率推理能力的本质研究最多的学科。

    过去 30 年来，普林斯顿大学的丹尼尔·卡尼曼（Daniel Kahneman,2002 年诺贝尔奖得主）及已故的阿莫斯·特维尔斯基（Amos Tversky）等心理学家的研究，彻底改变了我们对人类推理能力的认识

    # 10.1 常犯的统计推理错误

    • 未充分使用概率信息：人们在作出决策时，具体的单个事件的信息，往往会压倒较为抽象的概率信息（第四章中讨论的鲜活性问题）；在作出决策时，请考虑过去经验所提供的先验概率。

    如果每 1000 人中有 1 人携带人类免疫缺陷病毒（HIV），再假设有一种检查可以百分之百地诊断出真正携带该病毒的人。最后，假设这个检查有 5%的假阳性率。也就是说，这项检查在没有携带 HIV 的人中，也会错误地检测出有 5%的人是病毒携带者。假设我们随便找一个人来进行这项检查，结果呈阳性，表明此人为 HIV 携带者。假定我们对这个人的个人史或患病史一无所知，那么他真的是 HIV 携带者的概率是多少呢？

    # 11. 偶然性

    当一个特定的现象没有现成的系统性解释时，我们喜欢寻求结构的头脑不会因此停止动作，它会将无意义的理论强加在原本随机的数据上。

    当对行为进行预测时，千万不要忽视统计信息。统计预测也正确地预示着，当对人类行为进行预测时，错误和不确定性将始终存在。

    # 11.1 解释偶然性

    人们有解释偶然事件的倾向，这一现象在心理学的研究中被称为相关错觉。

    控制错觉： 人们相信个人力量可以影响偶然事件的结果。

    心理学也存在这样的倾向：研究者试图解释一切，希望其理论不仅能解释行为中系统的、非偶然的成分，还能解释任何细微的变异。这种倾向导致了不可证伪的信息学理论的泛滥。

    发生在我们个人生活中的罕见匹配往往对我们具有特殊的意义，因此我们尤其不愿意将其归因为偶然。

    # 11.2 临床预测与统计预测

    统计预测： 指一局从统计资料中得出的群体趋势所做的预测。 临床预测： 超越群体预测，对特定个体的结果做出百分之百准确的预测。

    我们在解释世界上发生的事件时，往往拒绝承认偶然因素的作用，这实际上会降低我们队现实世界的预测能力。

    一个例子：

    被试坐在两盏灯前（一红一蓝），实验者要求他们去预测每次测试时哪一盏会亮，被试要做多轮这样的实验，准确率越高，报酬越高。实际上左右的测试都是在 70%的次数亮红灯，30%亮蓝灯的条件下进行的，两种灯以随机顺序出现。

    如果你是被试，你观察到了红灯会亮 70%，蓝灯会亮 30%，你应该如何进行猜测？

    答案是永远猜红灯；

    人们常常会想去发觉随机因素后面的结构。

    # 12.不招人待见的心理学

    菜谱式读物：告诉你如何去使用某物，但对其基本的运作原理一概不谈的知识。

    本章提到了心理学的困境。

    • 心理学得到的尊重通常太少。
    • 心理学呈现给公众的形象（临床治疗师宣称自己具有独特的洞察人心的能力，而这种洞察力在研究证据方面却站不住脚。）通常又获得了过多的尊重。
    • 心理学领域的期刊太多，太多劣质期刊发表处于边缘水平的研究。
    • 心理学已成为单一的意识形态文化。

    过去十年来，大量准心理学概念被用来打压美国大学校园里的言论自由，而单一意识形态的文化阻碍了我们队这些概念进行质疑。

    我们许多个人心理学知识是菜谱式知识。我们做某些事是因为我们认为他们会导致其他人以某种方行事，或是因为我们相信这些事能帮助我们实现某些目标。这些都是菜谱式知识。科学心理学力图通过实证方法来检验菜谱式知识的有效性。

    许多伪科学已经发展成价值数百万美元的产业（星相预测、潜意识减肥、生物节律、辅助沟通术、通灵手术），他们之所以能蓬勃发展，利用的正是公众没有意识到关于行为的主张也可以用实证方法来检验这一事实。

    识别伪科学主张，伪科学主张有以下一些特征：

    • 总是提出一些似是而非的假设，是的主张免于被证伪。
    • 强调支持主张的证据，而忽略反驳齐主张的证据。
    • 总是将提供证据的任务强加给怀疑者而非支持者。
    • 过度依赖见证叙述来证实其主张；
    • 逃避同行评审所提供的审查。
    • 并非建立在已有科学的知识上。
