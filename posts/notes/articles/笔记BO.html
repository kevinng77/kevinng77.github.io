          # 贝叶斯优化

          Hyperparameters 超参的调整是 Deep Learning 较为耗时的一部分。比起 grid search 耗时耗力的暴力破解与 random search 的若有缘则相见。贝叶斯优化提供了一种相对有方向的搜索方案。

          # 算法大致思路

          SMBO 算法的本质，是建立并不断完善 hyper parameters (xxx)与 模型最优解 (f(x)f(x)f(x)) 间关系的过程。这个算法的终止由人为决定，比如限制算法运行时间，设置目标值等。

          # SMBO 算法解释

          在上述表格中 fff 为我们想要求出其超参的模型

          首先，选择几个超参方案 x1,x2,...x_1, x_2 ,...x1​,x2​,... 组成我们的超参集 XXX，并计算在这些超参组合下模型 fff 可以达到的最优分数 f(x)f(x)f(x)。因此对于每一个超参方案都会映射到一个最优分数。这些映射的集合便是 D={(x1,f(x1)),(x2,f(x2)),...}D = \{(x_1,f(x_1)),(x_2,f(x_2)),...\}D={(x1​,f(x1​)),(x2​,f(x2​)),...}

          而后我们对下面步邹迭代，直到达到目标条件：

          • 给定 prior DDD， 更新模型的 posterior P(f(x∗)∣x∗,D)P(f(x_*) | x_*,D)P(f(x∗​)∣x∗​,D)
          • 通过 posterior, 我们使用 acquisition function 求得一个新的超参方案 xt∈x∗x_t \in x_*xt​∈x∗​。不同的 axquisition funciton 能使我们选出的 xtx_txt​ 满足各种需求与假设。
          • 将 xtx_txt​ 带入到模型中，计算对应的 f(xt)f(x_t)f(xt​)， 并将 (xt,f(xt)(x_t,f(x_t)(xt​,f(xt​) 加入 DDD。这一步也是最耗费时间的一部。

          最后，当算法结束时。我们便可以从 DDD 中找到我们想要的最优超参

          高斯过程由均值函数 mmm 与核函数 (kernel function) kkk 决定，例如我们可以选择如下和函数：

          其中：

          Ky=κ(X,X)+σy2I=K+σy2IK∗=κ(X,X∗)K∗∗=κ(X∗,X∗) \begin{array}{c} \mathbf{K}_{y}=\kappa(\mathbf{X}, \mathbf{X})+\sigma_{y}^{2} \mathbf{I}=\mathbf{K}+\sigma_{y}^{2} \mathbf{I} \\ \mathbf{K}_{*}=\kappa\left(\mathbf{X}, \mathbf{X}_{*}\right) \\ \mathbf{K}_{* *}=\kappa\left(\mathbf{X}_{*}, \mathbf{X}_{*}\right) \end{array} Ky​=κ(X,X)+σy2​I=K+σy2​IK∗​=κ(X,X∗​)K∗∗​=κ(X∗​,X∗​)​

          # Acquisition Function

          如何找到下一个我们将要探索的点，是这整个算法的关键。与强化学习相似，我们希望我们的模型能够权衡 exploration 和 exploitation。

          # Expected Improvement

          EI 的假设是：我们观测到的 f(x)f(x)f(x) 并不存在误差项。

          EI 的 utility function 取决于我们的 objective function 的优化方向，一般来说，当我们使用 loss 等计算模型评分，此时我们希望 loss 越小越好，则：

          u(x)=max(0,f′−f(x)) u(x) = max(0, f' - f(x)) u(x)=max(0,f′−f(x))

          其中， f′=min(f)f' = min(f)f′=min(f)，即我们观测到的 DDD 中的模型最低分。

          因此

          xt=arg⁡max⁡x∈XE[u(x)∣x,D]=∫−∞f′(f′−f)N(f;μ(x),K(x,x))df=(f′−μ(x))Φ(f′;μ(x),K(x,x))+K(x,x)N(f′;μ(x),K(x,x)) x_t = \arg \max _{x \in \mathcal{X}}{E[u(x)|x,D]} \\ \begin{array}{l} =\int_{-\infty}^{f^{\prime}}\left(f^{\prime}-f\right) \mathcal{N}(f ; \mu(x), \mathcal{K}(x, x)) d f \\ =\left(f^{\prime}-\mu(x)\right) \Phi\left(f^{\prime} ; \mu(x), \mathcal{K}(x, x)\right)+\mathcal{K}(x, x) \mathcal{N}\left(f^{\prime} ; \mu(x), \mathcal{K}(x, x)\right) \end{array} xt​=argx∈Xmax​E[u(x)∣x,D]=∫−∞f′​(f′−f)N(f;μ(x),K(x,x))df=(f′−μ(x))Φ(f′;μ(x),K(x,x))+K(x,x)N(f′;μ(x),K(x,x))​

          若我们的优化对象为类似 R2R^2R2 的大而好指标。则：

          u(x)=max(0,f(x)−f′) u(x) = max(0,f(x) - f') u(x)=max(0,f(x)−f′)

          其中， f′=max(f)f' = max(f)f′=max(f)，即我们观测到的 DDD 中的模型最高分。

          因此：

          xt=arg⁡max⁡x∈XE[u(x)∣x,D]=arg⁡max⁡x∈X(μt−1(x)−f′)Φ(μt−1(x)−f′σt−1(x))+σt−1(x)ϕ(μt−1(x)−f′σt−1(x)) x_t = \arg \max _{x \in \mathcal{X}} E[u(x)|x,D]\\ =\arg \max _{x \in \mathcal{X}}\left(\mu_{t-1}(x)-f'\right) \Phi\left(\frac{\mu_{t-1}(x)-f'}{\sigma_{t-1}(x)}\right)+\sigma_{t-1}(x) \phi\left(\frac{\mu_{t-1}(x)-f'}{\sigma_{t-1}(x)}\right) xt​=argx∈Xmax​E[u(x)∣x,D]=argx∈Xmax​(μt−1​(x)−f′)Φ(σt−1​(x)μt−1​(x)−f′​)+σt−1​(x)ϕ(σt−1​(x)μt−1​(x)−f′​)

          其中 Φ\PhiΦ 和 ϕ\phiϕ 分别是 standard Gaussian distribution 的 cumulative distribution function（CDF）和 probability density function（PDF）。
